# Code-Usage
### Description
Visualizing personal GitHub programming language stats.

### Install Dependencies
Every dependency is listed in `package.json`.
```sh
$ npm install
```

### Run Locally
By default, the app should run at localhost:3000
```sh
$ npm start
```
or
```sh
$ ./bin/www
```
